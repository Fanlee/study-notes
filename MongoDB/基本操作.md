# MongoDB

## 显示所有数据库

```
show dbs
```

## 创建或者切换数据库

数据库存在则切换到指定数据库，不存在则创建，

```
use test
```

## 创建集合

MongoDB中的集合只有在插入一条数据后才会创建

```
db.createCollection('user')
```

## 查看集合

```
// 或者show tables
show collections

```
